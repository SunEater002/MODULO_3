Tipos de datos en Js

En Js existen 7 tipos de datos primitivos

  • String
  • Number
  • Boolean
  • Null
  • Undefined
  • Symbol
  • BigInt

Para saber el tipo de dato de una variable se puede usar el operador typeof

Por ejemplo:

            let nombre = "Juan";
            console.log(typeof nombre); // string

En el siguiente ejemplo se muestra el tipo de dato de una variable

            let nombre = "Juan";
            let edad = 25;
            let casado = false;
            let hijos = null;
            let mascotas = undefined;
            let simbolo = Symbol("mi simbolo");
            let bigInt = 1234567890123456789012345678901234567890n;

            console.log(typeof nombre); // string
            console.log(typeof edad); // number
            console.log(typeof casado); // boolean
            console.log(typeof hijos); // object
            console.log(typeof mascotas); // undefined
            console.log(typeof simbolo); // symbol
            console.log(typeof bigInt); // bigint
